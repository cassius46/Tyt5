📝 Her Dosyanın Amacı
=====================

1️⃣  index.html
   - Uygulamanın TEK HTML dosyası
   - Tüm CSS ve JavaScript içeride
   - Tarayıcıda açmanız yeterli
   - 26KB boyut

2️⃣  README.md
   - Proje açıklaması
   - Özellikler listesi
   - Kullanım talimatları
   - Teknik bilgiler

3️⃣  LICENSE
   - MIT Lisansı
   - Herkes kullanabilir demek
   - Ticari kullanım izni

4️⃣  package.json
   - Proje metadatası
   - npm için bilgiler
   - Versiyon, author vb.

5️⃣  .gitignore
   - Git'e hangi dosyaları yüklemeyeceğini söyler
   - OS ve IDE dosyaları hariç tutulur

6️⃣  GITHUB_UPLOAD.md
   - GitHub'a yükleme adımları
   - Komut satırı örnekleri
   - İpuçları

🚀 Hızlı Başlangıç
==================

1. index.html'i tarayıcıda açın (yerel olarak çalışır)
2. Terminal'de `git init` yapın
3. Dosyaları commit edin
4. GitHub'a push yapın
5. GitHub Pages'ten yayınlayın (opsiyonel)
