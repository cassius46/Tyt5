# 📊 TYT Takip Uygulaması

TYT (Temel Yeterlilik Testi) sınav sonuçlarını takip etmek ve çalışma ilerlemenizi ölçmek için tasarlanmış modern, hızlı ve kullanıcı dostu bir web uygulaması.

## 🎯 Özellikler

### 📈 Dashboard
- Son denemenin toplam net sonucunu görüntüleyin
- Hedefe ilerleme yüzdesini takip edin
- Ders bazında net gelişim grafiği
- Son 7 günlük çalışma saati analizi
- En çok yanlış yapılan konuları listeleyin

### ➕ Deneme Giriş
- Tarih ve deneme adı ile kayıt yapın
- Her ders için doğru/yanlış/boş sayılarını girin
- Otomatik net hesaplaması (Doğru - Yanlış/4)
- Yanlış yapılan konuları kaydedin
- Çalışma saatlerini giriş sırasında kaydedin

### ❌ Konu Yönetimi
- Derse göre yanlış konuları yönetin
- Konu sayısını takip edin
- Çalışılan konuları işaretleyin
- Konuları kolayca silin

### 🎯 Hedef Belirleme
- Her ders için hedef net belirleyin
- Gerçek performans vs hedef karşılaştırması
- İlerleme barı ile görsel takip

### ⏱️ Çalışma İstatistikleri
- Toplam çalışma süresi (saat:dakika)
- 14 günlük çalışma saati grafiği
- Ders bazında zaman dağılımı

### 📋 Geçmiş
- Tüm deneme sonuçlarını tabloda görüntüleyin
- Deneme adı ve tarihleriyle filtreleme
- Sonuçları silin

## 🚀 Hızlı Başlangıç

### Kurulum
1. Depoyu klonlayın:
```bash
git clone https://github.com/kullanıcıadı/tyt-takip.git
cd tyt-takip
```

2. `index.html` dosyasını bir web tarayıcısında açın

### Alternatif Yollar
- GitHub Pages üzerinden yayınlayın
- Herhangi bir web sunucusu ile kullanın
- Yerel dosya olarak açın (file://)

## 💾 Veri Depolama

Uygulama tüm verileri tarayıcının **localStorage** özelliğini kullanarak kaydeder:
- Veriler yerel olarak depolanır
- Tarayıcı temizlenmediği sürece verileri kaybetmezsiniz
- Farklı cihazlarda verileri açmak isterseniz export yapabilirsiniz

## 📝 Kullanım Örneği

1. **Deneme Giriş** sekmesine gidin
2. Tarih, deneme adı seçin
3. Her ders için doğru/yanlış/boş sayılarını girin
4. Yanlış konuları yazın (virgülle ayırın)
5. Çalışma saatlerini girin
6. **Kaydet** düğmesine basın
7. **Dashboard**'dan ilerlemenizi takip edin
